## **Project Goal**

This is an application that implements the principles of the C.R.U.D. cycle are defined as CREATE, READ/RETRIEVE, UPDATE, and DELETE, the application allows users to add recipes to the panel, update it and eventually delete them. A user needs to registrate to be able to add recipes.
